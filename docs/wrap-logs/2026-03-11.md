### 규칙 업데이트

1. **워크트리 머지 시 주의**: main에 untracked 파일이 있으면 `git merge`가 실패함. 해결법: main에서 해당 파일을 `git stash` 후 머지, 또는 `git add` 후 머지.
2. **`boolean && { ... }` 패턴 금지**: TypeScript strict 모드에서 TS2698 에러 발생. 항상 `condition ? { ... } : {}` 삼항 연산자 사용.
3. **`antigravity-cli -a`로 워크트리 병렬 작업 할당**: `antigravity-cli -a "프롬프트"` 패턴으로 비동기 에이전트 세션을 여러 개 생성하여 병렬 작업 가능.
