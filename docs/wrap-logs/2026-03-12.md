1. **`getMetaAccountForUser` 헬퍼 패턴**: 이 헬퍼는 내부적으로 `safeDecryptToken()`을 호출하여 반환하는 `accessToken`은 이미 복호화된 상태입니다. 새 API 라우트에서 이 헬퍼를 사용할 때 추가 복호화는 불필요하며, `const decryptedToken = metaAccount.accessToken` 패턴으로 명확성을 유지합니다.
2. **React 19 `set-state-in-effect` 경고**: `useEffect` 내 `setState` 대신 `useMemo` + `useRef`를 활용한 참조 비교 패턴을 사용합니다.
