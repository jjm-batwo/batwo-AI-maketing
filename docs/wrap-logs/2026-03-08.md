### 핵심 변경사항

- **[기능] RAG 시스템 전체 구현**: Supabase pgvector 기반 벡터 DB + text-embedding-3-small 임베딩 + cosine similarity 검색을 기존 Agentic Tool 아키텍처에 18번째 도구로 통합
- **[기능] KNOWLEDGE_QUERY 인텐트**: IntentClassifier/GuideQuestionService/FewShotExampleRegistry에 마케팅 지식 질문 분류 체계 추가
- **[기능] GPT-5 모델 전환**: 프로젝트 전체 AI 모델을 gpt-4o→gpt-5-mini, gpt-4o-mini→gpt-5-nano로 일괄 업그레이드
- **[버그] gpt-5-mini-mini 이중 치환 수정**: sed 순차 치환 시 gpt-4o-mini가 gpt-5-mini-mini로 잘못 변환된 문제 발견 및 수정 (kpiInsight.ts, KPIInsightsService.ts)

### 해결된 이슈

- pgvector 확장 설치 시 superuser 권한 필요 → 로컬 DB에 권한 부여로 해결
- `openai` npm 패키지 미설치 상태에서 임베딩 → 인라인 인터페이스 + fetch 기반 클라이언트로 해결
- KNOWLEDGE_QUERY 키워드 '방법', '어떻게' 등이 너무 포괄적 → 기존 GENERAL 테스트 깨짐 → 키워드 범위 축소
- sed 일괄 치환 시 gpt-4o→gpt-5-mini 후 gpt-4o-mini가 gpt-5-mini-mini로 이중 변환 → 수동 수정
