# 메타 광고 어트리뷰션 모델, 전환 창(Attribution Window), GA4 데이터 불일치 해결

## 1. 어트리뷰션(Attribution)이란?

어트리뷰션은 "이 전환(구매/가입)이 어떤 광고 덕분에 발생했는가?"를 판별하는 방법론이다. 메타의 어트리뷰션 시스템은 사용자가 광고를 **클릭(Click)** 또는 **조회(View)** 한 이후 일정 기간 내에 전환이 발생하면 해당 광고의 성과로 인정한다.

---

## 2. 2026년 기준 메타 전환 창(Attribution Window) 옵션

### 2026년 1월 12일 주요 변경 사항
- **7일/28일 조회(View-through) 창 완전 폐지**: 더 이상 데이터를 반환하지 않음
- **13개월 과거 데이터 제한**: 특정 필드(고유 지표, 시간별 데이터 등)에 대해 13개월 이전 데이터 조회 불가
- **참여 기반 어트리뷰션(Engage-through) 도입**: 기존 Engaged-View를 대체

### 현재 사용 가능한 전환 창 조합

| 전환 창 | 설명 | 적합한 상황 |
|---------|------|------------|
| **1일 클릭** | 광고 클릭 후 24시간 내 전환만 인정 | 즉각적 전환 (앱 설치, 충동 구매 상품) |
| **7일 클릭** (기본값) | 광고 클릭 후 7일 내 전환 인정 | 대부분의 이커머스 캠페인 (표준 권장) |
| **1일 클릭 + 1일 조회** | 클릭 후 24시간 또는 조회 후 24시간 내 전환 | 인지도 캠페인, 조회 기반 효과 측정 시 |
| **7일 클릭 + 1일 조회** | 클릭 후 7일 또는 조회 후 24시간 내 전환 | 가장 포괄적인 성과 추적 (ASC 캠페인용) |

### 기본 설정값
- **광고 세트 수준 기본값**: 7일 클릭 + 1일 조회
- 변경 시 광고 관리자(Ads Manager) → 광고 세트 → "어트리뷰션 설정"에서 수정 가능

---

## 3. 참여 기반 어트리뷰션(Engage-Through Attribution) — 2026년 신규

기존 "Engaged-View" 어트리뷰션을 대체한 새로운 모델이다.

### 핵심 변경 사항
- **동영상 참여 기준**: 기존 10초 → **5초**로 단축 (숏폼 콘텐츠 트렌드 반영)
- **클릭 어트리뷰션 재정의**: 이제 "진정한 링크 클릭(True Link Click)"만 인정. 좋아요, 공유, 저장 등의 인터랙션은 클릭이 아닌 **참여(Engage-through)로 재분류**
- GA4 등 서드파티 분석 도구와의 리포팅 정합성 향상이 목적

### SaaS 대응
- 영상 광고를 많이 집행하는 고객의 전환수가 갑자기 변동할 경우 → 어트리뷰션 모델 변경에 의한 것인지 실제 성과 변화인지를 구분하여 진단해야 함

## 4. "메타 vs GA4 전환수가 다릅니다" — 데이터 불일치 원인 및 해결 가이드

이것은 사용자가 가장 자주 묻는 질문 중 하나이다. 메타와 GA4(Google Analytics 4)의 전환수가 거의 항상 다르며, 이는 정상이다.

### 주요 원인
| 원인 | 세부 설명 |
|------|----------|
| **어트리뷰션 모델 차이** | 메타: 클릭/조회 기반, GA4: 라스트 클릭(Last Click) 기본. 같은 전환이라도 다른 채널에 기여를 귀속시킴 |
| **전환 창 차이** | 메타: 7일 클릭 + 1일 조회가 기본, GA4: 세션 기반 또는 30일 전환 창 |
| **조회 기반(View-through) 전환** | 메타는 광고를 "보기만" 하고 나중에 직접 방문하여 구매한 경우도 광고 성과로 잡음. GA4는 이를 "직접 유입(Direct)" 또는 "오가닉"으로 기록 |
| **크로스 디바이스 추적** | 메타는 로그인 기반으로 모바일 → PC 전환을 추적 가능. GA4는 쿠키 기반으로 디바이스 전환 시 추적 단절 |
| **광고 차단기/프라이버시** | 브라우저 광고 차단기가 GA4 스크립트를 차단하면 GA4에선 전환이 누락되지만, 메타 CAPI(서버사이드)에선 잡힘 |

### 해결 가이드
1. **"정답"은 없다**: 어느 한쪽이 맞고 틀린 것이 아니라, 측정 방법론의 차이임을 인지
2. **비교 기준 통일**: 메타 대시보드에서 "어트리뷰션 설정 비교" 기능으로 1일 클릭 vs 7일 클릭 수치를 나란히 확인
3. **UTM 파라미터 활용**: 모든 메타 광고 URL에 `?utm_source=facebook&utm_medium=paid&utm_campaign=캠페인명`을 반드시 추가하여 GA4에서 메타 트래픽만 정확히 필터링
4. **CAPI 연동 필수**: 서버사이드 전환 추적으로 브라우저 측 데이터 손실을 보완 → 메타 쪽 전환 정확도 상승
5. **메타 기준으로 ROAS 관리**: 메타 광고 최적화는 메타의 보고 데이터에 의존하므로, 입찰/예산 의사결정은 메타 대시보드 기준으로 하되, GA4는 전체 채널 간 비교 분석용으로 활용

### SaaS 진단 답변 예시
> "메타와 GA4의 전환수 차이는 어트리뷰션 모델의 차이에서 비롯됩니다. 메타는 광고를 '보기만 한 후(View-through)' 24시간 내 전환도 성과로 인정하지만, GA4는 '마지막 클릭'에 기여를 귀속시키므로 이 전환을 '직접 유입'으로 기록합니다. 정상적인 차이이며, 메타 보고서 내 '어트리뷰션 설정 비교' 기능으로 각 전환 창별 수치를 나란히 확인해 보시기를 권장합니다."
