# 크리에이티브 주도형 최적화: "크리에이티브가 곧 타겟팅이다"와 Entity ID의 수학적 원리

Andromeda 검색 엔진의 전면 도입으로 "크리에이티브가 곧 타겟팅이다(Creative is Targeting)"는 단순한 마케팅 격언을 넘어 **알고리즘을 지배하는 물리적이고 수학적인 법칙**으로 굳어졌다.

## 1. 엔터티 ID(Entity ID) — Andromeda의 소재 인식 방식

Andromeda가 개별 광고 소재를 인식하는 핵심 단위가 '엔터티 ID'이다.

### 과거 vs 현재
- **과거**: 동일한 이미지 배경에 헤드라인 텍스트만 10가지로 다르게 적어 업로드 → 알고리즘이 10개의 각기 다른 광고로 인식, A/B 테스트 수행
- **2026년**: Andromeda는 시각적 패턴이 유사한 이 10개의 이미지를 **단 1개의 Entity ID로 묶어버림**

### 치명적 결과
이는 방대한 광고 경매 시장에서 해당 캠페인이 **단 1장의 '입장권(Ticket)'만을 부여받는 것**과 동일하다. 메타 시스템은 시각적 유사도(Creative Similarity)가 높은 광고들이 피드에 도배되는 것을 극도로 혐오하기 때문이다.

### 해결 방안: 이질적인 Entity ID 10~15개 이상 공급
트래픽 볼륨을 확장하기 위해서는 시각적 구조와 맥락이 완전히 독립된 소재를 지속적으로 공급해야 한다:
- 사용자 생성 콘텐츠(UGC)
- 고품질 스튜디오 촬영본
- 텍스트 중심의 그래픽 이미지
- 창업자의 셀프 카메라
- 트렌디한 밈(Meme)

### SaaS 대응 로직
컴퓨터 비전(Computer Vision) 기술로 사용자가 등록한 광고 소재의 시각적 유사도를 수치화하여 판별. 예:
> "현재 업로드된 5개의 광고 소재는 모두 동일한 Entity ID로 분류되어 알고리즘의 노출 제한을 받을 위험이 85% 이상입니다. 화자가 변경되거나 배경 포맷이 완전히 다른 숏폼 영상을 즉시 추가하여 다양성 점수를 높이십시오."

## 2. 광고 피로도(Creative Fatigue) 관리

### 핵심 데이터
- 메타는 광고 관리자 대시보드에 'Creative Fatigue' 및 'Creative Similarity' 스코어 지표를 공식 도입
- 단일 소재가 동일 사용자에게 **3.5회 이상 지속 노출** 시 → 결과당 비용(CPA)이 **19% 이상** 가파르게 상승하는 페널티 패턴 확인 (2026년 퍼포먼스 마케팅 데이터 기준)

### 소재 교체 주기 기준 (2026년 공식 표준)
| 예산 규모 | 교체 주기 |
|----------|----------|
| 소규모 예산 | 최소 월 1회 |
| 대규모 예산 | 14~21일 주기 |

### 위험 시나리오
시각적 다양성이 심각하게 부족한 상태에서 일일 예산을 강제 증액할 경우 → Andromeda가 이미 해당 Entity ID에 피로한 사용자 군에게 억지로 광고 투입 → **CPM의 통제 불가능한 급등**

## 3. 동적 크리에이티브 최적화(DCO) 및 유연한 크리에이티브

### 권장 투입량
- 최소 **5개의 서로 다른 이미지 및 영상**
- **3개의 차별화된 헤드라인**
- **2개의 상세 설명**

메타 AI가 이 요소들을 해체하고 재조립하여 개별 사용자 맥락에 가장 부합하는 최적 조합을 **3~5일 내에** 자율적으로 찾아낸다.

---

## 4. 메타 생성형 AI 도구 (2025 칸 라이언즈 공개)

400만 명 이상의 광고주가 이미 활용 중인 도구들:

- **자동 브랜드 일관성(Automated Brand Consistency)**: 브랜드의 로고, 색상, 폰트를 학습하여 모든 생성형 변형 이미지에 일관 적용
- **이미지-투-비디오(Image-to-Video) 제너레이터**: 정적 제품 사진 여러 장을 매끄러운 다중 씬 비디오 광고로 자동 변환

SaaS 솔루션은 이러한 생성형 AI API를 통합하여 한 번의 클릭만으로 텍스트 및 시각적 변형 소재를 수십 개씩 자동 생성 → 캠페인에 재주입하는 End-to-End 자동화 루프를 완성해야 한다.
