# 메타 광고 정책 위반 유형, 계정 제재, 대응 방법 가이드

## 1. 광고 거부(Ad Rejection) — 가장 흔한 정책 위반 사유

메타는 AI 기반 자동 심사 시스템을 통해 광고를 검토한다. 아래 사유에 해당하면 광고가 즉시 거부되거나, 승인 후에도 소급 비활성화될 수 있다.

### 1-1. 금지 콘텐츠(Prohibited Content)
다음 콘텐츠가 포함된 광고는 **어떠한 경우에도 게재 불가**:
- 불법 제품/서비스 (마약류, 무기, 위조품 등)
- 성인 콘텐츠 (노출, 성적 암시)
- 차별적 표현 및 혐오 발언 (인종, 종교, 성별, 장애 등)
- 허위/오해의 소지가 있는 주장 ("이 약을 먹으면 100% 암이 낫습니다")
- 비현실적인 건강/다이어트 비포/애프터 이미지

### 1-2. 제한 콘텐츠(Restricted Content)
다음 콘텐츠는 광고 가능하나, **특별한 조건과 사전 허가(whitelisting) 필요**:
- 주류(알코올): 법적 음주 가능 연령 이상만 타겟팅 필수
- 온라인 도박/복권: 사전 허가 및 해당 국가 라이선스 필수
- 금융 서비스/암호화폐: 면책 조항 및 리스크 고지 필수
- 건강 보조제/의약품: 특정 국가별 규제 준수 필수
- 데이팅 서비스: 18세 이상 타겟팅 + 사전 허가

### 1-3. 개인 속성 직접 언급(Personal Attributes)
사용자의 개인적 특성을 직접적으로 언급하거나 암시하는 표현은 **즉시 거부**:
- ❌ "당신은 당뇨가 있으시죠?"
- ❌ "30대 싱글 남성을 위한..."
- ❌ "대출 연체 때문에 고민이시죠?"
- ✅ "당뇨 관리에 관심이 있는 분들을 위한 건강식 가이드"
- ✅ "합리적인 금융 솔루션을 찾는 분들에게"

### 1-4. 랜딩 페이지 문제
메타는 광고뿐 아니라 **광고의 도착 URL(랜딩 페이지)**도 심사한다:
- 랜딩 페이지 로딩 속도가 극도로 느린 경우
- 팝업이 과도하게 뜨는 경우
- 개인정보 처리방침(Privacy Policy)이 없는 경우
- 광고 내용과 랜딩 페이지 내용이 불일치하는 경우
- 깨진 링크(404 오류)

### 1-5. 시각적/텍스트 포맷 문제
- 이미지 내 과도한 텍스트 (2026년에도 알고리즘이 텍스트가 적은 이미지를 선호)
- 가짜 시스템 알림(예: "바이러스가 감지되었습니다!" 형태의 클릭베이트)
- 가짜 재생(Play) 버튼으로 클릭 유도
- 심각한 문법/맞춤법 오류
- 저품질 이미지

## 2. 계정 비활성화(Account Disablement) — 원인과 예방

### 2-1. 계정 비활성화 주요 원인
| 원인 | 설명 |
|------|------|
| **반복 정책 위반** | 같은 유형의 위반이 지속적으로 발생할 경우 |
| **의심스러운 계정 활동** | 다수의 IP/기기에서 로그인, 갑작스러운 예산 급증, 신규 계정 즉시 대규모 광고 집행 |
| **결제 문제** | 결제 실패 반복, 잔액 부족, 밴 된 계정의 결제수단 재사용 |
| **집행 우회 시도** | 정책 위반 광고를 다른 계정에서 재복제, 클로킹(Cloaking) 기법 사용 |
| **낮은 계정 신뢰도 점수** | 메타 AI가 계정 행동, 기기 디지털 핑거프린트, 과거 밴 기록을 분석하여 산정하는 신뢰도 점수 |
| **장기 미사용** | 광고 계정을 60일 이상 미사용 시 비활성화 위험 증가 |

### 2-2. 계정 보호를 위한 Best Practice
1. **워밍업(Warm-up)**: 신규 광고 계정은 즉시 대규모 예산을 넣지 말고, 소액($5~$10/일)부터 시작하여 1~2주간 점진적으로 증액
2. **결제수단 분리**: 밴 된 계정과 같은 신용카드를 절대 공유하지 않기
3. **2단계 인증 활성화**: 계정 보안 강화로 '의심스러운 활동' 감지 방지
4. **정기적 광고 집행**: 60일 이상 계정을 방치하지 않기
5. **정책 업데이트 뉴스레터 구독**: Meta Business Help Center에서 최신 정책 변경 사항 추적

## 3. 광고 거부 시 대응 프로세스

### 3단계 대응 절차
1. **원인 확인**: 광고 관리자 → 해당 광고 → "정책 이슈" 탭에서 거부 사유 확인
2. **수정 후 재심사 요청**: 문제가 된 텍스트/이미지/랜딩 페이지를 수정한 뒤 "심사 요청(Request Review)" 버튼 클릭
3. **이의 제기(Appeal)**: 오심이라 판단되면 Meta Business Support를 통해 이의 제기 가능 (보통 24~48시간 내 답변)

### SaaS 자동 진단 로직
> "업로드하신 광고 소재가 메타 정책 심사에서 거부되었습니다. 거부 사유: '개인 속성 직접 언급(Personal Attributes)'. 현재 헤드라인 '당신의 체중이 80kg을 넘으셨다면'이 사용자의 신체적 특성을 직접 언급하여 정책을 위반합니다. '건강한 체중 관리를 원하는 분들을 위한 가이드'와 같이 간접적 표현으로 수정하시면 승인될 가능성이 높습니다."
