# 데이터 인프라 및 성과 측정: CAPI, 1사 데이터, EMQ, UTIS, 가치 기반 입찰

## 1. 하이브리드 트래킹: 픽셀 + 전환 API(CAPI)

2026년은 서드파티 픽셀의 추적 효력이 사실상 완전히 상실된 프라이버시 최우선(Privacy-First)의 시대이다. 오직 하이브리드 트래킹 기술과 1사 데이터(1st-Party Data) 중심의 어트리뷰션 모델만이 생존을 담보한다.

### 필수 구현 사항
- 브라우저 기반 **메타 픽셀(Pixel)** + 서버사이드 추적 방식인 **전환 API(Conversions API, CAPI)**를 시스템 양단에 동시 구현하는 하이브리드 트래킹
- 이는 **권장 사항이 아닌 캠페인 집행의 절대적 필수 전제 조건**

### 핵심 성과 데이터
- 픽셀 단독 사용 대비 → CAPI 병행 시 **기여 전환수(Attributed Conversions) 평균 24% 상승**
- CAPI는 브라우저 광고 차단기(Ad-blocker)나 모바일 기기의 개인정보 보호 설정으로 인해 유실되는 전환 데이터를 서버 단위에서 완벽히 보완

## 2. 이벤트 매칭 품질(EMQ, Event Match Quality)

오프사이트 전환 이벤트를 실제 사용자의 메타 프로필과 정확히 매칭시키는 점수이다. **SaaS 시스템이 사활을 걸어야 하는 영역**이다.

### SaaS 시스템의 역할
- 사용자의 자사몰 데이터베이스에서 확보한 고객의 **해시화된 이메일, 전화번호, IP 주소, 브라우저 정보** 등의 1사 데이터를 암호화하여 메타 서버로 실시간 전송하는 백엔드 파이프라인 구축
- **EMQ 점수가 6.0 이하로 하락할 경우** → 데이터 전송 스크립트의 누락 여부를 즉각 스캔하고 복구하는 자가 치유(Self-healing) 기능 지원

## 3. UTIS (User True Interest Survey) 모델

2026년 1월 메타가 전격 도입한 새로운 성과 측정 방식이다.

### UTIS란?
- 메타가 사용자에게 특정 동영상이나 광고 콘텐츠가 자신의 실제 관심사와 얼마나 일치하는지를 **1~5점 척도로 직접 질문**하고 피드백 수집하는 매커니즘

### 정밀도 비교
| 측정 방식 | 정밀도 |
|----------|--------|
| 기존 영상 체류 시간 기반 추론 | 48.3% |
| **UTIS 모델** | **63.2%** |

### 핵심 영향
- UTIS 점수는 단순한 참고 지표가 아니라 **Lattice의 최종 랭킹 결정에 직접적인 가중치로 작용**하여 노출 우선순위를 뒤바꿈

## 4. 가치 기반 입찰(Value-Based Bidding)

### 트렌드 변화
- 과거: 무조건 최저 비용(Lowest Cost) 목표 입찰 → 2026년에 도태됨
- **현재**: 기대 ROAS(Return on Ad Spend)나 목표 CPA를 설정하여 시스템이 **전환 가치가 높은 진성 유저만을 골라 입찰**

### 경매 시스템의 총 가치(Total Value Score) 산정
메타 경매 시스템은 다음 3가지를 합산하여 총 가치를 산정:
1. 사용자의 **예상 전환 확률**
2. 광고주의 **입찰가**
3. 광고 자체의 **품질 및 관련성 평가 (Ad Quality Score)**

### 가치 규칙(Value Rules) — LTV 기반 입찰 가중치
메타의 '가치 규칙(Value Rules)'을 활용하면 단순 전환수가 아닌 **실질적인 고객 생애 가치(LTV)가 높은 오디언스에게 입찰 가중치를 부여**할 수 있다.

- 특정 고가치 기기(예: iOS/Android), 구매 전환율이 높은 지역 등의 조건으로 **입찰가를 최대 1,000%까지 자동 조정** 가능
- 전환 '수량'이 아닌 전환 '가치'를 우선시하므로 CPA가 다소 높더라도 실질 ROAS가 개선되는 효과
- SaaS 시스템은 사용자의 과거 구매 데이터를 분석하여 고가치 세그먼트를 자동 식별하고 가치 규칙 적용을 권장해야 함

### 페널티 메커니즘
- 하위 35%에 해당하는 **'평균 이하(Below Average)' 등급**을 받은 광고 소재는:
  - 경매 자체에서 **원천 배제**되거나
  - 동일 노출을 얻기 위해 **2~3배 더 높은 비용** 지불 필요

### SaaS 진단 로직
> "해당 소재가 UTIS 및 품질 평가에서 Below Average 등급을 받아 시스템으로부터 노출 페널티를 부여받았습니다. 즉시 게재를 중단하고 크리에이티브 품질을 개선하십시오."

### SaaS CAPI 연동 모니터링
- SaaS는 고객의 **CAPI 연동 여부를 필수적으로 체크**하고, 미연동 상태일 경우 즉시 경고 알림을 발행해야 함
- 브라우저 픽셀만 사용하는 고객에게는 데이터 누수 위험과 기여 전환수 24% 손실 가능성을 자동 진단하여 CAPI 연동을 강력 권고
